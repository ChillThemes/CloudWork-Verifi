A plugin to validate purchases using the Envato API

== Description ==

adds an extra field to the wp-login.php registration form that validates purchases using the Envato API. Also adds a shortcode [cw-verifi-registration] for registration on the frontend

== Installation ==

CloudWork Verifi is  easily installed automatically via the Plugins tab in your blogs admin panel, then it is configured with your Envato username and API key in Setting -> CW-Verifi

= Manual Installation =

1. Upload the `cw-verifi` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy!
